# assignment_techolution
Run the application by ```python3 main.py```

**The options available are -**
  1. Add Book
  2. List Books
  3. Search Books
  4. Update Book
  5. Delete Book
  6. Add User
  7. List Users
  8. Search Users
  9. Update User
  10. Delete User
  11. Checkout Book
  12. Check-in Book
  13. Exit

**NOTES:**
  
  **a.** The data for user and books is saved in ```users.json``` and ```books.json``` respectively.
  
  **b.** Logs are stored in ```library.log```.
